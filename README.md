# Aura GIF Editor

브라우저 기반 GIF / 스프라이트 편집 도구. 설치 없이 바로 사용 가능.

**[https://aura1905.github.io/gif-editor/](https://aura1905.github.io/gif-editor/)**

---

## 주요 기능

### 프레임 편집
- **드래그 앤 드롭** — GIF 파일을 끌어다 놓으면 바로 로드
- **멀티 문서 탭** — 여러 GIF를 동시에 열고 탭으로 전환
- **프레임 리스트** — 썸네일, 번호, 재생 시간 한눈에 확인
- **프레임 조작** — 삭제, 복사, 빈 프레임 삽입, 다중 선택 (Ctrl+클릭, Shift+클릭)
- **프레임 자동 축소** — 목표 프레임 수 입력하면 균등 간격으로 자동 조절
- **프레임 클립보드** — 탭 간 프레임 복사/붙여넣기 (Ctrl+C/V)

### 그리기 도구
- **연필 (Pencil)** — 현재 프레임에 픽셀 그리기
- **지우개 (Eraser)** — 픽셀을 투명으로 지우기
- **Undo/Redo** — 그리기 되돌리기/다시 실행 (최대 50단계)
- **팔레트** — GIF에서 자동 추출한 색상 팔레트 (좌측 고정 패널)
- **브러시 크기** — 1~32px 조절 가능 (`[` `]` 단축키)
- **브러시 커서** — 오버레이 방식으로 픽셀 단위 정확한 범위 표시

### 캔버스
- **줌인/줌아웃** — 마우스 휠 또는 Ctrl+/- (최대 1600%)
- **패닝** — 휠 버튼(미들 클릭) 드래그로 캔버스 이동
- **체커보드 배경** — 투명 영역을 시각적으로 확인
- **사이즈 변경** — 프리셋 (512, 256, 128, 96, 64) 또는 커스텀 입력
- **속도 조절** — 전체 프레임 속도 일괄 변경
- **미리보기 재생** — 편집 결과를 바로 확인

### 태그 시스템
- **프레임 태그** — 프레임 범위에 이름/색상 태그 지정 (idle, attack, run 등)
- **프리셋 태그** — 게임 애니메이션용 프리셋 + 자동 색상 매칭
- **Aseprite 호환** — 태그 정보가 .aseprite 내보내기에 반영

### 내보내기
- **GIF 다운로드** — 편집된 GIF 파일로 저장
- **Aseprite 내보내기** — RGBA 32bpp + zlib 압축 .aseprite 파일 생성

## 단축키

| 키 | 기능 |
|---|------|
| `Space` | 재생 / 일시정지 |
| `←` `→` | 이전 / 다음 프레임 |
| `Delete` | 선택한 프레임 삭제 |
| `Ctrl+Z` | 되돌리기 (Undo) |
| `Ctrl+Shift+Z` / `Ctrl+Y` | 다시 실행 (Redo) |
| `Ctrl+A` | 전체 선택 |
| `Ctrl+C` | 선택한 프레임 복사 |
| `Ctrl+V` | 프레임 붙여넣기 |
| `Ctrl+D` | 선택한 프레임 복제 |
| `B` | 연필(브러시) 도구 토글 |
| `E` | 지우개 도구 토글 |
| `[` `]` | 브러시 크기 감소 / 증가 |
| `마우스 휠` | 줌인 / 줌아웃 |
| `Ctrl++` `Ctrl+-` | 줌인 / 줌아웃 |
| `Ctrl+0` | 줌 리셋 (100%) |
| `휠 버튼 드래그` | 캔버스 이동 (패닝) |
| `?` | 단축키 도움말 |

## 프라이버시

- 모든 처리는 **브라우저 안에서** 수행됩니다
- 파일이 **서버로 전송되지 않습니다**
- 인터넷 연결 없이도 사용 가능 (오프라인 지원)

## 기술 스택

- HTML + CSS + JavaScript (순수 바닐라, 프레임워크 없음)
- [omggif](https://github.com/deanm/omggif) — GIF 디코딩/인코딩 (MIT License)
- 빌드 도구 없음 — HTML 파일을 직접 열면 바로 실행
- GitHub Pages로 배포

## 파일 구조

| 파일 | 역할 |
|------|------|
| `index.html` | SPA 레이아웃 (툴바, 프레임 패널, 캔버스, 모달) |
| `app.js` | 메인 애플리케이션 로직 (상태관리, UI, GIF 처리, 그리기 도구) |
| `style.css` | 다크 테마 UI 스타일 시스템 |
| `aseprite-encoder.js` | .aseprite 파일 바이너리 인코더 |
| `lib/omggif.js` | GIF 디코딩/인코딩 라이브러리 |
