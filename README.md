# 🎞️ GIF Editor

브라우저 기반 GIF 편집 도구. 설치 없이 바로 사용 가능.

🔗 **[지금 사용하기 → https://hanaura93.github.io/gif-editor/](https://hanaura93.github.io/gif-editor/)**

---

## ✨ 주요 기능

- 🖱️ **드래그 앤 드롭** — GIF 파일을 끌어다 놓으면 바로 로드
- 🎞️ **프레임 리스트** — 썸네일, 번호, 재생 시간 한눈에 확인
- ✂️ **프레임 편집** — 삭제, 복사, 다중 선택 (Ctrl+클릭, Shift+클릭)
- 🔢 **프레임 수 자동 축소** — 목표 프레임 수 입력하면 균등 간격으로 자동 조절
- 📐 **사이즈 변경** — 프리셋 (512, 256, 128, 96, 64) 또는 커스텀 입력
- ⏱️ **속도 조절** — 전체 프레임 속도 일괄 변경
- ▶️ **미리보기 재생** — 편집 결과를 바로 확인
- 💾 **다운로드** — 편집된 GIF 파일로 저장

## ⌨️ 단축키

| 키 | 기능 |
|---|------|
| `Space` | 재생 / 일시정지 |
| `←` `→` | 이전 / 다음 프레임 |
| `Delete` | 선택한 프레임 삭제 |
| `Ctrl+A` | 전체 선택 |
| `Ctrl+D` | 선택한 프레임 복사 |

## 🔒 프라이버시

- 모든 처리는 **브라우저 안에서** 수행됩니다
- 파일이 **서버로 전송되지 않습니다**
- 인터넷 연결 없이도 사용 가능 (오프라인 지원)

- HTML + CSS + JavaScript (순수 바닐라, 프레임워크 없음)
